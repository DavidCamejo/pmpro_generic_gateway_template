A collection of customizations useful when allowing users to Generic Gateway for Paid Memberships Pro levels.

== Description ==

A generic payment gateway template for devs.

== Installation ==

1. Upload the 'pmpro-generic-gateway' folder to the '/wp-content/plugins/' directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Change your Payment Settings to the "Generic Gateway" gateway and make sure to set the "Instructions" with instructions for how to Generic Gateway. Save.
1. Change the Payment Settings back to use your gateway of choice. Behind the scenes the Generic Gateway settings are still stored.
1. Edit your membership levels and set the "Generic Gateway Settings" for each level.

If you would like to change the wording from "Generic Gateway" to something else, you can use this custom code:
https://github.com/DavidCamejo
